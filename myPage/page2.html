CSS SPECIFICITY คืออะไร???

CSS Specificity เป็นคอนเซปต์การเลือก CSS Selector ที่มี "ความสำคัญ" เยอะที่สุด มาใช้กับ Element บนหน้าเว็บไซต์ของเรา

แล้ว Selector ของเรามีอะไรบ้างหละ?

Selector ของเราประกอบไปด้วย

  • Element Selector
  • Id Selector
  • Class Selector

    • Element Selector

    ก็คือ tag HTML ทั้งหลายนั้นแหละ โดยเราให้ค่าเท่ากับ 1 เช่น tag p ที่เป็น paragraph ตัวอย่างเช่น

                                p { 
                                color: red;
                                font-size: 2.4em; 
                            }

    Selector ใน block ข้างบนจะหมายความว่า ทุกๆ tag p ของเราจะ จะเป็นสีแดงและมีขนาดของ font เท่ากับ 2.4em

    Id Selector

    คือการเลือก element เราให้พิเศษมากขึ้น เช่นต้องการให้ tag p เป็นสีอื่น เราจะเขียนเริ่มต้นด้วย # โดยเราให้ค่าเท่ากับ 100

    การเขียน ID จะต้อง

  • ไม่เริ่มด้วยตัวเลข
  • ชื่อ id จะต้องไม่ซ้ำกับ id อื่นใน page เดียวกัน
  • id selector ของเราจะกับ element แต่ละตัวเท่านั้น
    •                                 #alert-center {
                                    color: red;
                                    text-align: center;
                                }
    Class Selector

    คือการเลือก element เราให้พิเศษมากขึ้น เราจะเขียนเริ่มต้นด้วย .(จุด) โดยเราให้ค่า = 10

                                    .center { 
                                    text-align: center; 
                                 }
    Grouping Selector

    นอกจาก Selector 3 แบบข้างบนแล้วยังมีอีกแบบนึงคือการ Group หรือรวมกลุ่มกันสำหรับ element ที่มี คุณสมบัติเหมือนกันกัน เช่น 

                            h1, h2, h3{
                            color: red;
                        }

    หมายความว่า tag h1, h2, h3 จะมีสีแดงเหมือนกันนั้นเอง


    ภาษา CSS เหมือนภาษาการเขียนโปรแกรมอื่น ๆ ที่อ่านโค้ดจาก บน – ล่าง นั้นหมายความว่าถ้าเขียน CSS Selector ซ้ำกัน 2 อัน อันล่างจะถูกนำไปใช้งานเพราะใหม่กว่า แต่ความพิเศษของ CSS คือ Selector ที่แตกต่างกันแต่ละ Selector จะมี “ความสำคัญ” ที่ไม่เหมือนกัน และเมื่อไหร่ที่ “ความสำคัญ” ต่างกัน ลำดับการเขียนโค้ดจะไม่มีผลทันที

    HTML: 

                        < class="box" id="redbox">

    CSS: 

                         .box { background: white; } /* ให้ .box มีพื้นหลังสีขาว*/
                    #redbox { background: red; } /* ให้ #redbox มีพื้นหลังสีแดง*/

    เราก็จะได้พื่นหลังสีแดงมาแบบนี้


    CSS Selector ทั้ง 2 อันของเรามีค่าความสำคัญที่แตกต่างกัน โดยค่าความสำคัญจะคำนวณจากการใช้ class, ID หรือการซ้อน Element หลาย ๆ ชั้นครับ ยิ่ง Selector นั้นเจาะจงเท่าไหร่ยิ่งมีค่าความสำคัญเยอะเท่านั้น ทีนี้ก็มาดูกันว่า CSS Selector ในตัวอย่างมีค่าความสำคัญเท่าไหร่กันบ้าง

    .box { } มีค่า = 10

    #redbox มีค่า = 100


    เพราะฉะนั้น CSS Property ใน #redbox { } (ค่า 100) จะถูกดึงมาแสดงผลทับของ .box { } (ค่า 10) เพราะมี ค่าความสำคัญมากกว่า นั่นเอง
